#tar

### Funktion des Pakets

- Erstellen, Extrahieren und Verwalten von Archivdateien (tarballs)
- Dateien zu Archiven hinzufügen, Inhalte auflisten oder vorhandene Archive aktualisieren

### Kompilierungsprozess (Kurz)

- Vorbereitung:

```bash
FORCE_UNSAFE_CONFIGURE=1
\

. /configure -- prefix=/usr
```

- Kompilierung:

```bash
make
```

- Testen:

```bash
make check
```

- Installation:

```bash
make install
make -C doc install-html docdir=/usr/share/doc/tar-1.35
```

- Hinweis: Ein Test (capabilities: binary store/restore) schlägt fehl, wenn selinux fehlt; Test wird übersprungen, falls Host-Kernel keine erweiterten Attribute/Security-Labels unterstützt

### Nach-der-Installation-Konfiguration

- Keine spezielle Konfiguration im LFS-Abschnitt angegeben
- Dokumentation installiert unter **/usr/share/doc/tar-1.35**

### Essenzielle Programme

- #tar
    
    - Programm: **tar**
        
    - Bedeutung für Systemintegration: 10/10 (Archivverwaltung, Backup, Wiederherstellung, Paket-Handling)
        
    - Absolut relevante Befehle und Nutzung (auswendig lernen):
        
        - Archiv erstellen (unkomprimiert):
        
        ```bash
        tar -cf archiv.tar datei1 datei2
        ```
        
        - Archiv entpacken (unkomprimiert):
        
        ```bash
        tar -xf archiv.tar
        ```
        
        - Inhalt eines Archivs anzeigen:
        
        ```bash
        tar -tf archiv.tar
        ```
        
        - Archiv erstellen mit gzip-Kompression:
        
        ```bash
        tar -czf archiv.tar.gz verzeichnis/
        ```
        
        - Archiv entpacken mit gzip-Kompression:
        
        ```bash
        tar -xzf archiv.tar.gz
        ```
        
        - Beim Entpacken in ein bestimmtes Verzeichnis:
        
        ```bash
        tar -xzf archiv.tar.gz -C /ziel/pfad
        ```
        
        - Einzelne Datei aus einem Archiv extrahieren:
        
        ```bash
        tar -xf archiv.tar pfad/zur/datei
        ```
        
        - Dateien zu bestehendem Archiv hinzufügen:
        
        ```bash
        tar -rf archiv.tar weitere_datei
        ```
