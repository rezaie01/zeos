#patch

### Funktion des Pakets

- Programm zum Modifizieren oder Erstellen von Dateien durch Anwenden einer Patch-Datei
- Patch-Dateien meist mit diff erzeugt; patch wendet Unterschiede an und erzeugt gepatchte Versionen

### Kompilierungsprozess (Kurz)

- Vorbereitung:
    
    ```bash
    . /configure -- prefix=/usr
    ```
    
- Kompilieren:
    
    ```bash
    make
    ```
    
- Testen:
    
    ```bash
    make check
    ```
    
- Installation:
    
    ```bash
    make install
    ```
    

### Nach-der-Installation-Konfiguration

- Keine zusätzliche Konfiguration im LFS-Abschnitt angegeben

### Essenzielle Programme

- #patch
    - Programm: **patch**
    - Zweck: Wendet Patch-Dateien auf Originaldateien an, erzeugt gepatchte Versionen
    - Wichtigkeit für Ziele: 5/10 (wichtig zum Verstehen von Quellcode-Wartung und Patching-Prozess; nicht unmittelbar systemkritisch für Erstkonfiguration)
