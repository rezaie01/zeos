# Kbd-2.8.0

- Paket enthält Keymaps, Konsolenfonts und Tastatur-Utilities
- Verwaltung von Tastaturbelegung, Konsolenanzeige und virtuellen Terminals

---

## Kompilierungsprozess

- Patch für Backspace/Delete-Tasten:
    
    ```bash
    patch -Np1 -i ../kbd-2.8.0-backspace-1.patch
    ```
    
- Entfernen von `resizecons`:
    
    ```bash
    sed -i '/RESIZECONS_PROGS=/s/yes/no/' configure
    sed -i 's/resizecons.8 //' docs/man/man8/Makefile.in
    ```
    
- Konfiguration:
    
    ```bash
    ./configure --prefix=/usr --disable-vlock
    ```
    
- Kompilieren:
    
    ```bash
    make
    ```
    
- Installation:
    
    ```bash
    make install
    ```
    
- Optional: Dokumentation installieren
    
    ```bash
    cp -R -v docs/doc -T /usr/share/doc/kbd-2.8.0
    ```
    

---

## Nach-der-Installation

- Verzeichnisse: `/usr/share/consolefonts`, `/usr/share/consoletrans`, `/usr/share/keymaps`, `/usr/share/unimaps`, `/usr/share/doc/kbd-2.8.0`
- Keymaps für bestimmte Sprachen ggf. extern beschaffen (z. B. Belarusisch)

## Essenzielle Programme und Nutzung

### #loadkeys

- Lädt Tastatur-Übersetzungstabellen
- Typische Nutzung:
    
    ```bash
    loadkeys de
    ```
    
- Aktiviert deutsches Tastaturlayout

### #dumpkeys

- Gibt aktuelle Tastatur-Übersetzungstabellen aus
- Typische Nutzung:
    
    ```bash
    dumpkeys
    ```
    
- Nützlich für Debugging von Keymaps

### #setfont

- Ändert Konsolen-Fonts (EGA/VGA)
- Typische Nutzung:
    
    ```bash
    setfont Lat2-Terminus16
    ```
    
- Anpassung der Schriftgröße und Zeichensätze

### #showconsolefont

- Zeigt aktuelles Konsolen-Font
- Typische Nutzung:
    
    ```bash
    showconsolefont
    ```
    

### #showkey

- Zeigt Scancodes, Keycodes und ASCII-Codes gedrückter Tasten
- Typische Nutzung:
    
    ```bash
    showkey
    ```
    
- Wichtig für Fehleranalyse bei Tastaturproblemen

### #setkeycodes

- Lädt Scancode-zu-Keycode-Mapping
- Typische Nutzung:
    
    ```bash
    setkeycodes e05b 112
    ```
    
- Nützlich für Sondertasten oder ungewöhnliche Keyboards

### #setleds

- Setzt Tastatur-LEDs (NumLock, CapsLock, ScrollLock)
- Typische Nutzung:
    
    ```bash
    setleds +num
    ```
    

### #chvt

- Wechselt auf anderes virtuelles Terminal
- Typische Nutzung:
    
    ```bash
    chvt 2
    ```
    

### #fgconsole

- Zeigt aktive Konsole
- Typische Nutzung:
    
    ```bash
    fgconsole
    ```
    

### #openvt

- Startet Programm auf neuem VT
- Typische Nutzung:
    
    ```bash
    openvt bash
    ```
    

### #kbd_mode

- Zeigt oder setzt Tastaturmodus
- Typische Nutzung:
    
    ```bash
    kbd_mode -a
    ```
    

### #kbdrate

- Setzt Wiederholrate und Verzögerung der Tastatur
- Typische Nutzung:
    
    ```bash
    kbdrate -r 30 -d 250
    ```
    

### #unicode_start / #unicode_stop

- Aktiviert bzw. deaktiviert Unicode-Modus für Konsole
- Typische Nutzung:
    
    ```bash
    unicode_start
    unicode_stop
    ```

## Fokus für Systemintegration

- #loadkeys → Tastaturlayouts setzen, wichtig für internationale Systeme
- #dumpkeys / #showkey → Debugging bei fehlerhaften Eingaben oder Sondertasten
- #setfont / #showconsolefont → Anpassung der Konsolenanzeige, nützlich für Server ohne GUI
- #chvt / #openvt / #fgconsole → Verwaltung von virtuellen Terminals, wichtig für Mehrbenutzerumgebungen
- #setkeycodes → Integration von Spezialhardware oder nicht-standardisierten Tastaturen
- #setleds → Statusanzeigen für Benutzer, Skripte oder Diagnose
- #kbdrate → Optimierung der Eingabe für bestimmte Arbeitsumgebungen
