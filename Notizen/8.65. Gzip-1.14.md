# Gzip-1.14

- Paket für Komprimierung und Dekomprimierung von Dateien
- Nutzt Lempel-Ziv (LZ77) Algorithmus
- Standardformat: `.gz`

---

## Kompilierungsprozess

- Konfiguration:
    
    ```bash
    ./configure --prefix=/usr
    ```
    
- Kompilieren:
    
    ```bash
    make
    ```
    
- Testen:
    
    ```bash
    make check
    ```
    
- Installation:
    
    ```bash
    make install
    ```
    

---

## Nach-der-Installation

- Keine zusätzliche Konfiguration erforderlich
- Programme stehen direkt zur Verfügung

---

## Essenzielle Programme und Nutzung

### #gzip

- Komprimiert Dateien
- Typische Nutzung:
    
    ```bash
    gzip datei.txt
    ```
    
- Ergebnis: `datei.txt.gz`
- Optionen:
    - `-d` → Dekomprimieren
    - `-c` → Ausgabe auf stdout
    - `-r` → rekursiv in Verzeichnissen

### #gunzip

- Dekomprimiert `.gz` Dateien
- Typische Nutzung:
    
    ```bash
    gunzip datei.txt.gz
    ```
    
- Ergebnis: Originaldatei wiederhergestellt

### #zcat

- Dekomprimiert und gibt Inhalt auf stdout aus
- Typische Nutzung:
    
    ```bash
    zcat datei.txt.gz
    ```
    
- Nützlich für Pipes und direkte Verarbeitung

### #zgrep

- Führt `grep` auf komprimierten Dateien aus
- Typische Nutzung:
    
    ```bash
    zgrep "Suchbegriff" datei.txt.gz
    ```
    
- Erspart vorherige Dekomprimierung

### #zless

- Anzeige von komprimierten Dateien mit `less`
- Typische Nutzung:
    
    ```bash
    zless datei.txt.gz
    ```
    

### #zmore

- Anzeige von komprimierten Dateien mit `more`
- Typische Nutzung:
    
    ```bash
    zmore datei.txt.gz
    ```
    

### #zcmp

- Vergleich von komprimierten Dateien mit `cmp`
- Typische Nutzung:
    
    ```bash
    zcmp datei1.gz datei2.gz
    ```
    

### #zdiff

- Vergleich von komprimierten Dateien mit `diff`
- Typische Nutzung:
    
    ```bash
    zdiff datei1.gz datei2.gz
    ```
    

### #zforce

- Erzwingt `.gz` Endung bei komprimierten Dateien
- Typische Nutzung:
    
    ```bash
    zforce datei
    ```
    

### #znew

- Konvertiert alte `.Z` Dateien ins `.gz` Format
- Typische Nutzung:
    
    ```bash
    znew datei.Z
    ```
    

### #gzexe

- Erstellt selbstentpackende ausführbare Dateien
- Typische Nutzung:
    
    ```bash
    gzexe script.sh
    ```
    
- Ergebnis: komprimiertes, aber direkt ausführbares Skript

---

## Fokus für Systemintegration

- #gzip / #gunzip → Standardwerkzeuge für Dateiarchivierung und Transfer
- #zgrep → Analyse von Logdateien direkt im komprimierten Zustand
- #zcat → Verarbeitung komprimierter Daten in Pipes
- #zless / #zmore → schnelle Sichtprüfung von komprimierten Logs
- #zcmp / #zdiff → Vergleich von Backups oder Konfigurationsdateien im komprimierten Format
- #gzexe → Platzsparende Bereitstellung von Skripten
