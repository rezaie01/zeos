#texinfo

### Funktion des Pakets

- Programme zum Lesen, Schreiben und Konvertieren von Info-Seiten
- Erzeugt/manipuliert Dokumentation in Info-, Text- und HTML-/PDF-Formaten

### Kompilierungsprozess (Kurz)

- Fix für Perl-5.42+ Warnung:

```bash
sed 's/! $output_file eq/$output_file ne/' -i tp/Texinfo/Convert/ *. pm
```

- Vorbereitung:

```bash
. /configure -- prefix=/usr
```

- Kompilierung:

```bash
make
```

- Testen:

```bash
make check
```

- Installation:

```bash
make install
```

- Optional: TeX-Komponenten installieren:

```bash
make TEXMF=/usr/share/texmf install-tex
```

- Hinweis: TEXMF legt die Wurzel des TeX-Baums fest

### Nach-der-Installation-Konfiguration

- Falls Info-Index veraltet, /usr/share/info/dir neu erzeugen:

```bash
pushd /usr/share/info
rm -v dir
for f in *
do install-info $f dir 2>/dev/null
done
popd
```

- Dokumentation und Bibliotheken unter /usr/share/texinfo und /usr/lib/texinfo

### Essenzielle Programme und Nutzung

- #info
    
    - Zweck: Lesen von Info-Seiten; umfassendere Dokumentation als man-Seiten
    - Wichtigkeit: 6/10 (nützlich für Paket- und Tool-Dokumentation; nicht systemkritisch)
    - Relevante Nutzung: Navigation in Info-Seiten; keine spezifischen Befehlsoptionen zwingend zum Auswendiglernen für LFS-Grundlagen
- #install-info
    
    - Zweck: Info-Seiten in den Info-Index aufnehmen / Index aktualisieren
    - Wichtigkeit: 6/10 (wichtig, um Info-Index konsistent zu halten)
    - Relevante Nutzung: Wird in der Regel von Paketinstallationsscripts verwendet; direktes Ausführen zur Wartung des Info-Index (siehe oben)
- #makeinfo (texi2any)
    
    - Zweck: Texinfo-Quellen in Info, Text, HTML etc. übersetzen
    - Wichtigkeit: 4/10 (relevant bei eigener Dokumentationserstellung)
- Weitere Programme (kurz): pdftexi2dvi; pod2texi; texi2any; texi2dvi; texi2pdf; texindex
    
    - Zweck: verschiedene Konvertierungs- und Formatierungswerkzeuge für Texinfo/TeX
    - Wichtigkeit: 3–4/10 (nur bei Bedarf zur Dokumentenverarbeitung)
