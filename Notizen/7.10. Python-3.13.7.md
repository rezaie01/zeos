# 7.10. Python-3.13.7
Das Python 3-Paket enthält die Python-Entwicklungsumgebung. Sie ist nützlich für **objektorientierte Programmierung**, das **Schreiben von Skripten**, das **Prototyping großer Programme** und die **Entwicklung ganzer Anwendungen**. Python ist eine interpretierte Computersprache.
- **Geschätzte Bauzeit (`Approximate build time`):** 0.5 SBU
- **Erforderlicher Speicherplatz (`Required disk space`):** 546 MB

---

## 7.10.1. Installation von Python

><font color="#938953">HINWEIS:</font>
> *Es gibt zwei Paketdateien, deren Namen mit dem Präfix „python“ beginnen. Diejenige, die entpackt werden soll, ist Python-3.13.7.tar.xz (man beachte den Großbuchstaben am Anfang).*

**Vorbereitung zur Kompilierung von Python:**

```bash
./configure --prefix=/usr \
    --enable-shared \
    --without-ensurepip \
    --without-static-libpython
```

**Bedeutung der Configure-Optionen (Gelb hervorgehobene Punkte):**
- `--enable-shared`: **Dieser Schalter verhindert die Installation statischer Bibliotheken.**
- `--without-ensurepip`: **Dieser Schalter deaktiviert den Python-Paket-Installer, der zu diesem Zeitpunkt nicht benötigt wird.**
- `--without-static-libpython`: **Dieser Schalter verhindert das Bauen einer großen, aber unnötigen, statischen Bibliothek.**

**Kompilierung des Pakets:**


```bash
make
```
><font color="#938953">HINWEIS:</font>
>*Einige Python 3-Module können jetzt nicht gebaut werden, da die Abhängigkeiten noch nicht installiert sind. Zum Beispiel wird bei dem `ssl-Modul` eine Meldung ausgegeben, dass Python ein **`OpenSSL 1.1.1`** oder neuer benötigt. Die Meldung sollte ignoriert werden. Es muss nur sichergestellt werden, dass der Top-Level-Befehl make nicht fehlgeschlagen ist. Die optionalen Module werden jetzt nicht benötigt und sie werden in Kapitel 8 gebaut.*

**Installation des Pakets:**


```bash
make install
```
