## Groff-1.23.0

### Zweck des Pakets

- Text- und Bildformatierungssystem
- Frontend für `troff` mit Postprozessoren für verschiedene Ausgabeformate (PDF, PS, HTML, DVI, Text)
- Enthält zahlreiche Präprozessoren für Tabellen, Gleichungen, Diagramme und Schriftarten

### Kompilierung

- Erwartet Umgebungsvariable `PAGE` für Standard-Papiergröße (`letter` oder `A4`)
- Konfiguration:

```bash
PAGE=<paper_size> ./configure --prefix=/usr
```

- Kompilieren:

```bash
make
```

- Testen:

```bash
make check
```

- Installation:

```bash
make install
```

### Nach-der-Installation-Konfiguration

- Papiergröße kann nachträglich gesetzt werden:

```bash
echo "A4" > /etc/papersize
```

oder

```bash
echo "letter" > /etc/papersize
```

### Essenzielle Programme und Nutzung

- #groff
    
    - Frontend für das gesamte System
    - Führt `troff` aus und wählt passenden Postprozessor
    - Beispiel:
        
        ```bash
        groff -Tpdf file.roff > file.pdf
        ```
        
- #troff
    
    - Kernprogramm zur Textformatierung
    - Normalerweise über `groff` aufgerufen
    - Direktnutzung selten, aber Basis für alle Ausgaben
- #nroff
    
    - Emuliert klassisches `nroff` mit `groff`
    - Ausgabe für reine Text-Terminals
    - Beispiel:
        
        ```bash
        nroff -man file.1 | less
        ```
        
- #eqn
    
    - Präprozessor für mathematische Gleichungen
    - Eingebettet in `troff`-Dateien
    - Beispiel:
        
        ```bash
        eqn file.eqn | groff -Tpdf > eqn.pdf
        ```
        
- #tbl
    
    - Präprozessor für Tabellen
    - Beispiel:
        
        ```bash
        tbl file.tbl | groff -Tpdf > table.pdf
        ```
        
- #pic
    
    - Präprozessor für Diagramme
    - Beispiel:
        
        ```bash
        pic file.pic | groff -Tps > diagram.ps
        ```
        
- #grops
    
    - Postprozessor für PostScript-Ausgabe
    - Beispiel:
        
        ```bash
        groff -Tps file.roff > file.ps
        ```
        
- #gropdf
    
    - Postprozessor für PDF-Ausgabe
    - Beispiel:
        
        ```bash
        groff -Tpdf file.roff > file.pdf
        ```
        
- #grotty
    
    - Postprozessor für Terminal-Ausgabe (ASCII, ANSI)
    - Beispiel:
        
        ```bash
        groff -Tascii file.roff
        ```
        
- #roff2pdf
    
    - Direkter Konverter von roff-Dateien zu PDF
    - Beispiel:
        
        ```bash
        roff2pdf file.roff > file.pdf
        ```
        
- #roff2html
    
    - Direkter Konverter von roff-Dateien zu HTML
    - Beispiel:
        
        ```bash
        roff2html file.roff > file.html
        ```
        
- #pdfroff
    
    - Wrapper für PDF-Erstellung mit erweiterten Optionen
    - Beispiel:
        
        ```bash
        pdfroff -ms file.roff > file.pdf
        ```
        
- #pdfmom
    
    - Speziell für Dokumente mit `mom`-Makros
    - Beispiel:
        
        ```bash
        pdfmom file.mom > file.pdf
        ```
