#make

## Funktion des Pakets

- Werkzeug zur Steuerung der Kompilierung von Programmen und Dateien aus Quellcode
- Erkennt automatisch, welche Teile neu kompiliert werden müssen
- Führt die entsprechenden Kommandos aus

## Kompilierungsprozess

- Vorbereitung:
    
    ```bash
    ./configure --prefix=/usr
    ```
    
- Kompilierung:
    
    ```bash
    make
    ```
    
- Testen:
    
    ```bash
    chown -R tester .
    su tester -c "PATH=$PATH make check"
    ```
    
- Installation:
    
    ```bash
    make install
    ```
    

## Nach-der-Installation-Konfiguration

- Keine zusätzliche Konfiguration erforderlich

## Essenzielle Programme

### #make

- **make**
    - Automatisiert den Bauprozess von Software
    - Liest Regeln aus `Makefile`
    - Bestimmt Abhängigkeiten und führt nur notwendige Kompilierungen aus
    - Wichtige Nutzung:
        - `make` → startet Kompilierung basierend auf `Makefile`
        - `make target` → baut nur angegebenes Ziel
        - `make clean` → entfernt erzeugte Dateien, Vorbereitung für Neu-Kompilierung
        - `make install` → installiert erzeugte Programme ins System
