#markupsafe

### Funktion des Pakets

- Python-Modul zur sicheren Behandlung von XML/HTML/XHTML-Markup-Strings

### Kompilierungs-/Installationsprozess (exakt wie im Buch)

- Erstelle Wheel:

```bash
pip3 wheel -w dist --no-cache-dir --no-build-isolation --no-deps $PWD
```

- Hinweis: Dieses Paket enthält keine Test-Suite
- Installation:

```bash
pip3 install --no-index --find-links dist Markupsafe
```

### Nach-der-Installation-Konfiguration

- Keine zusätzliche Konfiguration im LFS-Abschnitt angegeben
- Installiertes Verzeichnis: /usr/lib/python3.13/site-packages/MarkupSafe-3.0.2.dist-info
