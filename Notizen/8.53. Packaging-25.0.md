### 2. Kompilierungsprozess

#### **Build (Wheel-Erstellung)**

- Kompilierung durch Erstellung eines **Wheel-Archivs** (`.whl`):
    ```
    pip3 wheel -w dist --no-cache-dir --no-build-isolation --no-deps $PWD
    ```
    
    - `pip3 wheel`: Baut das Wheel-Archiv.
        
    - `-w dist`: Speichert das Wheel im Verzeichnis `dist`.
        
    - `--no-cache-dir`: Verhindert das Kopieren in den `pip`-Cache.
        

#### **Installation**

- Installation des lokal erstellten Wheels:
    ```
    pip3 install --no-index --find-links dist packaging
    ```
    
    - `--no-index`: Ignoriert Online-Paket-Repositorys (PyPI).
        
    - `--find-links dist`: Sucht das Wheel im lokalen `dist`-Verzeichnis.
        
    - `--no-build-isolation`, `--no-deps`: Gew√§hrleisten, dass keine Online-Quellen f√ºr Build- oder Laufzeitabh√§ngigkeiten genutzt werden.
