# 7.12. Util-linux-2.41.1
Das Util-linux-Paket enthält **verschiedene Dienstprogramme** (`miscellaneous utility programs`).

- **Geschätzte Bauzeit (`Approximate build time`):** 0.2 SBU
    
- **Erforderlicher Speicherplatz (`Required disk space`):** 192 MB
    

---

## 7.12.1. Installation von Util-linux

**Vorbereitung:**

Die **FHS empfiehlt** (`FHS recommends`) die Verwendung des Verzeichnisses `/var/lib/hwclock` **anstelle** (`instead`) des **üblichen** (`usual`) `/etc`-Verzeichnisses für die `adjtime`-Datei. Erstellen Sie dieses Verzeichnis mit dem folgenden Befehl:


```bash
mkdir -pv /var/lib/hwclock
```

**Vorbereitung zur Kompilierung von Util-linux:**


```bash
./configure --libdir=/usr/lib \
    --runstatedir=/run \
    --disable-chfn-chsh \
    --disable-login \
    --disable-nologin \
    --disable-su \
    --disable-setpriv \
    --disable-runuser \
    --disable-pylibmount \
    --disable-static \
    --disable-liblastlog2 \
    --without-python \
    ADJTIME_PATH=/var/lib/hwclock/adjtime \
    --docdir=/usr/share/doc/util-linux-2.41.1
```
> **<font color="#9bbb59">ERKLÄRUNG</font>:**
> - **`ADJTIME_PATH=/var/lib/hwclock/adjtime`**:
>     - Dies legt den **Speicherort** der Zeitaufzeichnungs-Informationen über die Hardware-Uhr **gemäß dem FHS fest**.
>     - Dies ist für diese temporäre Umgebung nicht zwingend erforderlich, erstellt jedoch eine Datei an einem anderen Ort, die beim Bau des endgültigen Util-linux-Pakets nicht überschrieben oder entfernt werden würde.
> - **`--libdir=/usr/lib`**:
>     - Dieser Schalter stellt sicher, dass die `.so`-Symlinks auf die **freigegebene Bibliotheksdatei** (`shared library file`) direkt im Verzeichnis `/usr/lib` liegen (und nicht in `/usr/lib64`).
> - **`--disable-*`**:
>     - **Diese Schalter verhindern Warnungen über fehlende Komponenten, die in LFS oder noch nicht installiert sind, erforderlich sind.**
> - **`--without-python`**:
>     - **Dieser Schalter deaktiviert die Verwendung von Python.** Er vermeidet den Versuch, unnötige Bindungen zu bauen.
> - **`runstatedir=/run`**:
>     - Dieser Schalter legt den Speicherort des Sockets fest, der von `uuidd` und `libuuid` verwendet wird.
> 
---

**Kompilierung des Pakets:**


```bash
make
```

**Installation des Pakets:**


```bash
make install
```
