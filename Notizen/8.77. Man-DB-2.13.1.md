#man-db

### Funktion des Pakets

- Werkzeuge zum Finden, Formatieren und Anzeigen von Manualseiten (man-pages).
- Erzeugt/aktualisiert whatis- und man-Cache.

---

### Kompilierungsprozess (exakt wie im Buch)

```bash
. /configure \
--prefix=/usr \
--docdir=/usr/share/doc/man-db-2.13.1 --sysconfdir=/etc \
--disable-setuid \
--enable-cache-owner=bin \
--with-browser=/usr/bin/lynx \
--with-vgrind=/usr/bin/vgrind \
--with-grap=/usr/bin/grap \
--with-systemdtmpfilesdir= \
--with-systemdsystemunitdir=
```

```bash
make
```

```bash
make check
```

```bash
make install
```

---

### Nach-der-Installation-Konfiguration

- Man-DB erwartet man-Seiten unter /usr/share/man.
- Nicht-UTF-8 legacy Manpages: Man-DB nimmt Zeichenkodierung laut Verzeichnis /usr/share/man/<ll> an (Tabelle im Buch).
- Für Index/Cache regelmäßig mandb ausführen nach Installation oder Änderungen:

```bash
mandb
```

### Essenzielle Programme und Befehle (Analyse Ziele: Systemintegration / Anfänger → auswendig lernen)

- #man
    
    - Wichtigkeit: 10/10
    - Zweck: Manualseite formatieren und anzeigen.
    - Wichtige Nutzung (auswendig):
        - man programm
- #mandb
    
    - Wichtigkeit: 10/10
    - Zweck: Erzeugt/aktualisiert die whatis- und man-Cache-Datenbanken.
    - Wichtige Nutzung (auswendig):
        - mandb
- #whatis / #apropos
    
    - Wichtigkeit: 9/10
    - Zweck: Kurze Beschreibungen aus der whatis-Datenbank suchen.
    - Wichtige Nutzung (auswendig):
        - whatis befehl
        - apropos suchbegriff
- #manpath
    
    - Wichtigkeit: 8/10
    - Zweck: Zeigt MANPATH oder berechneten Suchpfad anhand von man.conf und Umgebung.
    - Wichtige Nutzung (auswendig):
        - manpath
- #catman
    
    - Wichtigkeit: 6/10
    - Zweck: Erzeugt/aktualisiert vorformatierte Manualseiten (für Performance).
    - Wichtige Nutzung (auswendig):
        - catman
- #man-recode / #lexgrog / #accessdb
    
    - Wichtigkeit: 5/10
    - Zweck: man-recode konvertiert Encodings; lexgrog zeigt Kurzinfo; accessdb dump des whatis.
    - Wichtige Nutzung (kenntnisbasiert): man-recode bei Encoding-Problemen; lexgrog für schnelle Zusammenfassung.
