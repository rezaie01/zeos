### 2. Kompilierungsprozess

#### **Vorbereitung und Build (Wheel-Erstellung)**

- Der Build-Prozess nutzt `pip3` zur Erstellung eines **Wheel-Archivs**:
    ```
    pip3 wheel -w dist --no-cache-dir --no-build-isolation --no-deps $PWD [cite: 162]
    ```
    
    - `wheel`: Baut das Wheel-Archiv f√ºr das Paket2.
        
    - `-w dist`: Speichert das erstellte Wheel im lokalen `dist`-Verzeichnis3.
        
    - `--no-cache-dir`: Verhindert das Kopieren in den `pip`-Cache (`/root/.cache/pip`)4.
        

#### **Installation**

- Installation des erstellten Wheels:
    ```
    pip3 install --no-index --find-links dist flit_core [cite: 164]
    ```
    
- **Wichtige Optionen f√ºr LFS/Offline-Installation:**
    
    - `--no-index`, `--no-deps`: Verhindern das Abrufen von Dateien aus dem Online-Repository (PyPI)5.
        
    - `--find-links dist`: Weist `pip` an, die Wheel-Archive im `dist`-Verzeichnis zu suchen6.
